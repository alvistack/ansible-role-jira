Ansible Role for Atlassian Jira Installation.

## Requirements

This role require Ansible 2.9 or higher.

This role was designed for:

  - Ubuntu 18.04/19.10/20.04
  - Debian 10
  - RHEL/CentOS 7/8
  - openSUSE Leap 15.1
